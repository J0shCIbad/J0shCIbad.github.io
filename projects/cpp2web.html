Cpp2Web 

Project Description:

 A C++ web framework that abstracts web technologies and allows the creation of web pages purely with C++. The framework comes with a multithreaded server program to serve the web pages as well as the capability to compile static websites written with Cpp2Web, into a series of HTML / CSS / JS files.

Technologies and Skills used:

LanguagesTechnologies
  • C++
  • HTML / CSS / JS
  • TCP/IP Socket Programming
  • POSIX Multithreading
